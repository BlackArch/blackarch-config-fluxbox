## Description

BlackArch Linux configuration files for the fluxbox window manager.

## Installation

`pacman -S blackarch-config-fluxbox`

## Configuration

Copy the following files:

1. `cp -a /usr/share/blackarch/config/fluxbox/* /usr/share/fluxbox`
